# 投资组合持仓表 Markdown 格式化（1992-1995年）- 产品需求文档

## 概述
- **摘要**：将 1992-1995 年共 4 份巴菲特致股东信中的纯文本投资组合持仓表转换为标准 Markdown 表格格式，提升阅读体验和数据可读性。
- **目的**：解决纯文本表格在不同设备上排版错乱、难以阅读的问题，确保数据完整性和展示一致性。
- **目标用户**：阅读和研究巴菲特股东信的读者、投资者、研究者。

## 目标
- 将 1992-1995 年股东信中所有纯文本投资组合表转换为标准 Markdown 表格
- 确保所有数字数据与原文完全一致，无遗漏或错误
- 表头根据实际列内容和单位设计，清晰标注单位（千美元/百万美元）
- 表格前后保留空行，符合 Markdown 排版规范

## 非目标（范围外）
- 不修改表格之外的任何原文内容
- 不修改已经是标准 Markdown 表格的内容
- 不对数据进行任何计算或修改（如单位转换、数值调整）
- 不添加任何分析、评论或解释性内容
- 不处理 1985-1991 年的文件（已在之前完成）

## 背景与上下文
- 项目为巴菲特股东信学习网站（bamangBOOK），核心原则是"学习原点"，原文内容必须完整保留
- 之前已完成 1985-1991 年共 7 份股东信的投资组合表转换
- 表格样式通过 MarkdownContent.tsx 组件统一渲染，支持暗色模式、响应式布局、斑马纹等
- 1995 年单位从千美元变为百万美元，需在表头中明确标注

## 功能需求
- **FR-1**：识别并转换 1992 年股东信中的普通股投资组合表（1个）
- **FR-2**：识别并转换 1993 年股东信中的普通股投资组合表（1个）
- **FR-3**：识别并转换 1994 年股东信中的普通股投资组合表（1个，结构较复杂）
- **FR-4**：识别并转换 1995 年股东信中的两个表格（普通股投资表 + 可转换优先股表）
- **FR-5**：根据每年实际列数动态设计表头，包含单位标注
- **FR-6**：确保公司名称完整保留（包含"-新进"、"-增持"、"-减持"等后缀）

## 非功能需求
- **NFR-1**：数据准确性：所有数字必须与原文完全一致，拆分正确率 100%
- **NFR-2**：结构完整性：每行列数一致，分隔符行与数据行管道符数量匹配
- **NFR-3**：原文完整性：表格外内容不做任何修改
- **NFR-4**：格式规范性：表格前后有空行，符合 Markdown 标准

## 约束
- **技术**：直接编辑 Markdown 文件，不使用脚本或工具批量处理
- **业务**：原文内容不可修改，仅格式转换
- **依赖**：无外部依赖，仅需文件编辑能力

## 假设
- 1992-1994 年表格单位为千美元（与 1991 年一致）
- 1995 年表格单位为百万美元（从数字格式可推断）
- 表格第一列为股份数量，最后一列为单价/成本/股，中间为公司名称、成本、市值
- 公司名称中可能包含数字（如"GEICO"），需结合上下文正确拆分
