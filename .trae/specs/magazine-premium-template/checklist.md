# Checklist

- [x] 有杂志式报头栏（刊头标记 + 装饰线）
- [x] 大标题使用无衬线体，字号 >= 28px
- [x] 大标题有冲击力/开页感
- [x] 小标题使用无衬线体，字号 17-18px，字重 600
- [x] 正文使用衬线体（Georgia + Noto Serif SC）
- [x] 正文颜色为近黑色（#1a1a1a）
- [x] 行高 >= 1.9
- [x] 段间距 >= 22px
- [x] 两侧留白 >= 24px
- [x] 区块间距 >= 32px
- [x] 引用块/Pull Quote 风格（放大字号 + 装饰线，无色块）
- [x] 无任何红色背景色块
- [x] 红色仅用于报头装饰、标题、强调标记
- [x] 表格极简风格（浅灰分隔线、灰标签）
- [x] 底部有杂志风格结束标记
- [x] 整体视觉感受像一本杂志
