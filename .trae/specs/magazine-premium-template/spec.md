## ADDED Requirements

### Requirement: 杂志风视觉语言

排版应呈现高端杂志的视觉质感。

#### Scenario: 杂志式头部
- **WHEN** 用户查看文章顶部
- **THEN** 应有一个杂志风格的品牌栏/报头，包含细装饰线和一个刊头标记（如小圆点或短横线）
- **THEN** 品牌栏与标题之间应有充足留白

#### Scenario: 标题处理
- **WHEN** 用户查看大标题
- **THEN** 标题应像杂志封面或专题开页一样有冲击力——大字号（28px+），粗字重
- **THEN** 标题下方可搭配一条细装饰线或小副标题区域，制造杂志"开页感"
- **THEN** 主标题使用无衬线体，与正文衬线体形成对比

#### Scenario: 文章正文
- **WHEN** 用户阅读正文
- **THEN** 正文使用高品质衬线体（Georgia + Noto Serif SC），字号 16-17px
- **THEN** 行高 1.9，段距充分（22px+）
- **THEN** 正文颜色为深灰/近黑（不一定是纯黑，可用 `#1a1a1a`）

#### Scenario: 章节分隔
- **WHEN** 用户从一个章节进入下一个章节
- **THEN** 章节之间应有明显的视觉分隔——可使用加大小标题间距、或一个装饰性分隔元素
- **THEN** 小标题应像杂志专题的"小标题"一样清晰但不抢眼

#### Scenario: 引用块/Pull Quote
- **WHEN** 用户看到引用/客户证言
- **THEN** 引用块应像杂志的 pull quote——放大字号（17-18px），灰色或红色文字，左右留白，可加左侧装饰线
- **THEN** 无色块，纯文字处理

#### Scenario: 数据可视化
- **WHEN** 用户看到数据对比/表格
- **THEN** 应像杂志信息图一样清晰——极简表格，浅灰分隔线，标签灰字
- **THEN** 关键数据可用稍大字号或强调色突出

#### Scenario: 文章底部
- **WHEN** 用户滚动到文章底部
- **THEN** 应有一个杂志风格的"结束标记"——如一个刊尾符号（■ 或 ●）或一条装饰线

### Requirement: 色彩体系

色彩应服务于"杂志感"。

#### Scenario: 主色
- **THEN** 主色为大红色 `#CC0000`，仅用于：报头装饰、标题（可选）、pull quote 装饰线、关键强调标记
- **THEN** 红色不得用于正文、不得用于背景色块、不得用于大面积边框

#### Scenario: 中性色
- **THEN** 正文：`#1a1a1a`
- **THEN** 辅助文字（标签、引用、脚注）：`#888` 或 `#666`
- **THEN** 分割线/边框：`#e0e0e0` 或更淡
- **THEN** 背景：纯白 `#fff`

### Requirement: 留白与呼吸感

杂志的核心是留白。

#### Scenario: 间距
- **THEN** 两侧留白 >= 24px
- **THEN** 段间距 >= 22px
- **THEN** 区块间距（标题与正文之间）>= 32px
- **THEN** 报头与标题之间 >= 20px

#### Scenario: 字体层次
- **THEN** 大标题：无衬线，28px，letter-spacing: 1px
- **THEN** 小标题：无衬线，17-18px，字重 600
- **THEN** 正文：衬线，16-17px，行高 1.9
- **THEN** 引用：衬线，17-18px，斜体或灰色
- **THEN** 标签/注释：无衬线或无特殊处理，14px
