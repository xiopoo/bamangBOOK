# Checklist

- [x] 标题上方有一条细红装饰线（2px 大红色）
- [x] 大标题使用大红色，无粗重边框
- [x] 大标题使用无衬线体，字号 >= 24px
- [x] 小标题使用黑色/深灰色，无红色，无边框
- [x] 正文使用衬线体（Georgia + Noto Serif SC）
- [x] 正文颜色为近黑色（#111 或 #1a1a1a）
- [x] 行高 >= 1.9
- [x] 段间距 >= 20px
- [x] 两侧留白 >= 24px
- [x] 无任何红色背景色块
- [x] 引用块无背景色，使用极细深灰左边框
- [x] 表格极简风格，无突兀背景色
- [x] 底部有细红装饰线，与顶部呼应
- [x] 整体视觉感受有高级感/杂志感
