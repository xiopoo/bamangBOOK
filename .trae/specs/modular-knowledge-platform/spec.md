# 模块化价值投资知识平台 - Product Requirement Document

## Overview
- **Summary**: 基于已抓取的 对标站点 内容，构建一个独立、原创的模块化价值投资知识平台。以巴菲特致股东的信为核心栏目，预留扩展接口支持后续集成芒格思想库、拆书、AI聊天等功能模块。
- **Purpose**: 建立一个可扩展、可维护的价值投资知识生态系统，摆脱对原网站的依赖，形成自有品牌的知识平台
- **Target Users**: 价值投资学习者、巴菲特/芒格思想研究者、中文投资社区用户

## Goals
- 对已抓取的 75 个页面内容进行系统性清理，移除所有原网站品牌标识和版权信息
- 构建模块化网站架构，巴菲特为核心模块
- 预留扩展接口，支持未来无缝集成"芒格长期阅读库"、"拆书"、"AI聊天"等模块
- 保持内容的完整性和知识图谱的交叉链接结构
- 实现响应式设计，支持移动端访问

## Non-Goals (Out of Scope)
- 不包含需要登录认证的私有内容
- 不实现实时数据同步/更新机制
- 不包含支付系统或电商功能

## Background & Context
- 已从 `对标站点 成功抓取 75 个页面，包括 35 封合伙人信、60 封伯克希尔股东信、49 个投资概念、61 家公司档案、7 位关键人物
- 原网站由"邦比快跑"运营，包含其品牌标识、版权声明、推广链接等
- 需要构建自有品牌的知识平台，支持未来扩展更多价值投资相关内容

## Functional Requirements

### FR-1: 内容清理
- **FR-1.1**: 移除所有"巴菲特知识库"品牌标识，替换为自有品牌名
- **FR-1.2**: 移除所有"by 邦比快跑"作者信息
- **FR-1.3**: 移除所有 bbseed.com 推广链接
- **FR-1.4**: 移除所有"完整版 PDF"商业推广
- **FR-1.5**: 移除"请我喝杯咖啡"/donate 捐赠链接
- **FR-1.6**: 替换网站标题、描述为自有内容
- **FR-1.7**: 移除百度统计、百度推送、Cloudflare Analytics 等第三方脚本
- **FR-1.8**: 移除微信公众号 QR 码及关注引导

### FR-2: 模块化架构
- **FR-2.1**: 核心模块 `buffett/` 包含所有巴菲特信件、概念、公司、人物内容
- **FR-2.2**: 预留模块 `munger/` 目录结构，首页显示"即将上线"占位
- **FR-2.3**: 预留模块 `book-reviews/` 目录结构
- **FR-2.4**: 预留模块 `ai-chat/` 路由入口
- **FR-2.5**: 模块注册中心 - 统一管理模块路由、导航项、状态
- **FR-2.6**: 导航栏根据模块注册动态生成，新增模块自动出现在导航

### FR-3: 内容展示
- **FR-3.1**: 首页展示全局统计数据（信件数、概念数、公司数、人物数）
- **FR-3.2**: 信件详情页保留原文结构化内容
- **FR-3.3**: 知识卡片页展示概念定义、核心要义、实战案例
- **FR-3.4**: 公司档案页展示投资历程、巴菲特评价
- **FR-3.5**: 人物档案页展示关系网络

### FR-4: 导航与搜索
- **FR-4.1**: 左侧导航栏支持分类浏览（合伙人信、股东信、概念、公司、人物）
- **FR-4.2**: 导航栏动态显示已注册模块
- **FR-4.3**: 实现站内全文搜索功能
- **FR-4.4**: 响应式设计，移动端导航折叠

## Non-Functional Requirements
- **NFR-1**: 网站为纯静态站点，可部署至 GitHub Pages / Vercel / Cloudflare Pages
- **NFR-2**: 页面加载时间 < 3s
- **NFR-3**: 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- **NFR-4**: 模块间完全解耦，新增模块不修改已有模块代码
- **NFR-5**: 代码使用模块化架构（ES Module 或组件化框架）

## Constraints
- **Technical**: 使用 HTML + CSS + JavaScript（无重型框架依赖），或采用轻量级 SSG（如 Astro/Hugo/VitePress）
- **Content**: 所有内容来源于已抓取的 75 个页面
- **Deployment**: 静态站点托管

## Assumptions
- 用户拥有已抓取内容的合法使用权利
- 新品牌名称将在实施阶段确定

## Acceptance Criteria

### AC-1: 内容清理完成
- **Given**: 原始抓取的 HTML 文件
- **When**: 运行内容清理脚本后
- **Then**: 所有原网站品牌标识、版权信息、推广链接、第三方脚本均被移除
- **Verification**: `programmatic`

### AC-2: 模块化架构可用
- **Given**: 网站构建完成
- **When**: 访问各模块页面
- **Then**: 巴菲特模块内容完整；芒格等预留模块显示"即将上线"占位
- **Verification**: `human-judgment`

### AC-3: 新增模块无需修改核心代码
- **Given**: 模块系统已完成
- **When**: 新增一个模拟模块注册
- **Then**: 只需在模块注册配置中添加条目，导航和路由自动更新
- **Verification**: `programmatic`

### AC-4: 巴菲特模块内容完整
- **Given**: 网站部署完成
- **When**: 浏览所有巴菲特模块页面
- **Then**: 98 封信件、49 个概念、61 家公司、7 个人物内容完整可访问
- **Verification**: `human-judgment`

### AC-5: 响应式设计正常
- **Given**: 网站在各种设备上访问
- **When**: 调整浏览器窗口尺寸
- **Then**: 导航栏自动折叠/展开，内容布局自适应
- **Verification**: `human-judgment`
