# 财务报表表格化整理 - 产品需求文档

## Overview
- **Summary**: 将内容文件中以纯文本形式存在的财务报表（损益表、资产负债表等）整理为标准 Markdown 表格格式，确保数据结构清晰、列对齐、便于阅读和对比分析。
- **Purpose**: 解决 OCR/转录过程中财务报表格式丢失的问题，将压缩成纯文本的财务数据恢复为结构化表格，提升财务数据的可读性和专业性。
- **Target Users**: 深度阅读财务内容的投资者、研究员，以及需要对比分析财务数据的读者。

## Goals
- 修复「所有者收益」vs「自由现金流」文章中的 2 个复杂财务报表（损益表 + 资产负债表）
- 修复 1982-1993 年代股东信中的"报告收益主要来源"类表格
- 确保所有财务报表数据完整、列对齐、结构清晰
- 保持原始数据的准确性，不修改任何数字或文字

## Non-Goals (Out of Scope)
- 不添加或修改任何财务数据，只调整格式
- 不处理已经是标准 Markdown 表格格式的报表
- 不处理低置信度的疑似报表（避免误判）
- 不涉及样式层面的修改（表格样式已在前几轮标准化）
- 不重写或润色报表周围的正文内容

## Background & Context
- 前几轮已修复了：表格重复表头、列数不匹配、尾部空列、表格数据行纯文本化（简单数字表格）、股东信标题提取等问题
- 本轮发现更深层的格式问题：复杂财务报表（损益表、资产负债表）完全以纯文本形式存在
- 这类报表的特点是：
  1. 有明确的中英文财务术语（收入/Revenues、毛利/Gross Profit 等）
  2. 多列数据对比（如公司 O vs 公司 N、1987 年 vs 1986 年）
  3. 包含小计、总计、脚注标记等复杂结构
  4. 所有数据被压缩成一整段纯文本

## Functional Requirements
- **FR-1**: 修复「所有者收益」vs「自由现金流」中的损益表（Scott Fetzer 公司 O vs 公司 N）
- **FR-2**: 修复「所有者收益」vs「自由现金流」中的资产负债表（Scott Fetzer 公司 O vs 公司 N）
- **FR-3**: 修复 1982-1993 年股东信中的"报告收益主要来源"表格（约 12 封）
- **FR-4**: 确保所有修复后的表格列数一致、数据对齐
- **FR-5**: 保留脚注标记（(1)(2)(3)...）和报表单位说明（千美金、千美元等）

## Non-Functional Requirements
- **NFR-1**: 数据准确性——修复后的表格数字、文字与原文完全一致，无任何修改
- **NFR-2**: 结构完整性——报表的层级结构（总收入、小计、总计等）清晰可辨
- **NFR-3**: 可读性——表格在桌面端和移动端均能正常显示，移动端支持横向滚动
- **NFR-4**: 可回滚性——所有修改可通过 Git 历史追溯和回滚

## Constraints
- **Technical**: 基于现有 Markdown 渲染管线，不引入新依赖
- **Business**: 内容必须忠实于原文，不得随意增删
- **Dependencies**: 依赖前几轮已完成的表格样式标准化

## Assumptions
- 财务报表虽然被压缩为纯文本，但各数据项之间有可识别的分隔特征
- 股东信中"报告收益主要来源"类表格的结构在不同年份之间基本一致，可以批量处理
- 脚注标记（(1)(2)等）紧跟在相关数字后面，可以保留在对应单元格中

## Acceptance Criteria

### AC-1: 损益表修复完成（Scott Fetzer）
- **Given**: 「所有者收益」vs「自由现金流」文章中的损益表纯文本
- **When**: 整理为标准 Markdown 表格后
- **Then**: 表格包含正确的行项目（收入、销售成本、毛利、经营收益、净利润等）和两列数据（公司 O、公司 N），所有数字准确无误
- **Verification**: `human-judgment`
- **Notes**: 需包含脚注标记，保留"-"前缀表示扣减项

### AC-2: 资产负债表修复完成（Scott Fetzer）
- **Given**: 「所有者收益」vs「自由现金流」文章中的资产负债表纯文本
- **When**: 整理为标准 Markdown 表格后
- **Then**: 表格包含正确的行项目（现金、应收、存货、固定资产、负债、股东权益等）和两列数据（公司 O、公司 N），所有数字准确
- **Verification**: `human-judgment`
- **Notes**: 资产、负债、权益三大类需清晰可辨

### AC-3: 股东信"报告收益主要来源"表格修复完成
- **Given**: 1982-1993 年股东信中的收益来源纯文本表格
- **When**: 整理为标准 Markdown 表格后
- **Then**: 每个表格包含业务板块行（保险承保、保险投资、喜诗糖果等）和年份列，数据完整对齐
- **Verification**: `human-judgment`
- **Notes**: 约 12 封股东信，结构相似

### AC-4: 数据完整性
- **Given**: 所有修复后的财务报表
- **When**: 对比原文和修复后的表格
- **Then**: 所有数字、文字、脚注标记、正负号完全一致，无任何丢失或错误
- **Verification**: `human-judgment`

### AC-5: 浏览器端正确渲染
- **Given**: 修复后的内容页面
- **When**: 在浏览器中访问
- **Then**: 财务报表以表格形式正确显示，行列对齐，数据清晰可读
- **Verification**: `human-judgment`

### AC-6: 暗色模式兼容
- **Given**: 修复后的财务报表
- **When**: 切换到暗色模式
- **Then**: 表格边框、背景、文字颜色在暗色模式下显示正常，对比度足够
- **Verification**: `human-judgment`
