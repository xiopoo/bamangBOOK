# 校对格式规范

## 一、标题规范

| 级别 | Markdown | 用途 | 格式 |
|---|---|---|---|
| 一级标题 | `#` | 仅用于文档标题 | `# 巴菲特致合伙人的信 1957` |
| 二级标题 | `##` | 主要章节 | `## 业绩回顾` |
| 三级标题 | `###` | 子章节 | `### 道指对比` |

**规则：**
- 一级标题仅用作文档标题，与文件名含义一致
- 文档正文内部不再重复使用一级标题
- 二级标题以下与正文之间空一行

## 二、术语规范

### 人名规范
格式：中文名（英文名）

| 中文名 | 英文名 | 备注 |
|---|---|---|
| 沃伦·巴菲特 | Warren Buffett | 全文中名统一，无需每处附英文 |
| 沃伦·E·巴菲特 | Warren E. Buffett | 首次出现时用全名 |
| 查理·芒格 | Charlie Munger | |
| 本杰明·格雷厄姆 | Benjamin Graham | |

**规则：** 同一自然段内首次出现外国人名时，应标注英文名，后续同段内可省略。

### 公司名规范
格式：中文名（英文名）

| 中文名 | 英文名 |
|---|---|
| 伯克希尔·哈撒韦 | Berkshire Hathaway |
| 道琼斯工业平均指数 | Dow Jones Industrial Average |
| 道指 | (可简称为"道指") |

### 数字格式
- 千分位：英文逗号分隔，如 `1,234,567`
- 百分比：统一使用 `XX.X%` 格式，如 `22.2%`
- 金额：美元统一使用 `$` 符号，如 `$1,000,000`

### 日期格式
`YYYY 年 MM 月 DD 日`，如 `1962 年 1 月 24 日`

## 三、段落规范

- 段落之间空一行
- 引用块使用 `>` 标记
- 无序列表使用 `-`，有序列表使用 `1.` `2.` `3.`

## 四、表格规范

```markdown
| 列1 | 列2 | 列3 |
|---|---|---|
| 内容1 | 内容2 | 内容3 |
```

**规则：**
- 使用标准 Markdown 表格语法
- 表头与数据行之间用 `|---|---|` 分隔（列数与表头一致）
- 列数必须保持一致
- 表格前后各空一行
- 表格内避免多余空格和冗余列

## 五、第三方内容处理规则

**定义：** 凡在 PDF 原版中不存在的内容，均为"第三方内容"。

**包括但不限于：**
- 编者分析、评论
- 知识链接
- 要点拆解
- 编者制作的财务数据汇总表（如 BRK 多年利润表、资产负债表）
- 背景注释、编者按

**处理方式：**
1. 用 `---` 分隔线与原文隔开
2. 置于文档末尾
3. 标题注明"编者注"

```markdown
原文内容...

---

## 编者注

（第三方内容置于此处）
```

## 六、已识别的差异示例

### 示例 1：股东信插入第三方分析（berkshire_1965）

**文件对比：**
- 网页版：`content/letters/berkshire_1965-巴菲特致股东信.md`
- PDF 版：`content/pdf-documents-formatted/巴菲特致股东的信_1965.md`

**网址版信件原文末尾（签名行）后插入了以下第三方内容：**
1. "伯克希尔的诞生和崛起滚雪球：由于比约定的回购价格少了 1/8 美元..."——背景故事分析
2. "巴菲特：如果你在投资时情绪化..."——巴菲特语录
3. "真实情况真的是这样吗？巴菲特自 1962 年开始购买伯克希尔..."——长篇编者分析
4. BRK 利润表 1954-1964、BRK 利润表 1964-1972、BRK 资产负债表 1954-1964、BRK 资产负债表 1964-1972 等编者整理的财务数据表
5. "巴菲特 1970 年代开始专注经营性业务"——编者结论

**问题类型：** 第三方内容未与原文分隔，直接堆砌在信件正文之后。

**整改要求：** 上述内容应移至 `---` 分隔线后，统一归入"编者注"部分。

### 示例 2：合伙契约被改写后又修复（partnership_1956）

**文件对比：**
- 网页版：`content/partnership/partnership_1956-巴菲特致合伙人信.md`
- PDF 版：`content/pdf-documents-formatted/巴菲特合伙契约_1956.md`

**差异分析：**

| 维度 | 网页版（partnership_1956） | PDF 版（巴菲特合伙契约_1956） |
|---|---|---|
| 标题 | `# 1956 有限合伙协议` | `# 巴菲特合伙契约 1956` |
| 摘要 | 开头有引用块摘要 > 巴菲特职业生涯起点... | 无摘要，直接进入正文 |
| 段落结构 | 用 `**第一条**`、`**第二条**` 等编号，清晰分条 | 条款合并为自然段落，如全行 15 混排一气 |
| 人名格式 | 统一 `中文名（英文名）` 格式 | 英文名混在中文中，无统一格式 |
| 表格 | 人物列表和出资额用表格呈现 | 部分内容以表格呈现，部分以文本内嵌 |
| 注释 | 表格中附带"家庭关系"栏 | 无家庭关系栏 |
| 末尾备注 | 无 | PDF 末尾有"备注"段说明背景 |

**问题类型：**
- 网页版曾被改写为带有编者分析内容的"摘要式"版本（引用块开头），后又部分修复
- 网页版增加了"家庭关系"注释列，属于 PDF 中没有的第三方内容
- 两版标题不统一

**整改要求：** 以 PDF 版为基准，网页版应保持原文结构，编者分析入"编者注"。

### 示例 3：表格格式问题（partnership_1961-annual）

**文件对比：**
- 网页版：`content/partnership/partnership_1961-annual-巴菲特致合伙人信.md`
- PDF 版：`content/pdf-documents-formatted/巴菲特致合伙人的信_1961_年报.md`

**差异分析：**

**表格 1（业绩表格）：**
- 网页版表头：`| 年份 | 全年管理合伙人账户数量 | 合伙公司收益率 | 道指收益率 |  |`（有末尾空列）
- PDF 版表头：`| 年份 | 全年管理合伙人账户数量 | 合伙公司收益率 | 道指收益率 |  |`（有末尾空列）
- 网页版数据行：`| 1957年 | 3 | 10.40% | -8.40% |`
- PDF 版数据行：`| 1957 | 年 | 3 | 10.40% | -8.40% |`（"年"被错误放在第二列）

**表格 2（LP 收益率表格）：**
- 网页版表头行出现重复：第 32-33 行 `| 年份 | LP | 合伙人收益率 | 道指收益率 |` 和 `| 年份 | 合伙人收益率 | 道指收益率 |`（列数不一致）
- PDF 版也有类似问题：第 40 行 `|---|---|---|---|` 列数多，后续行少列

**表格 3（基金对比表格）——网页版特有：**
- 表头出现三次重复，列数不一致（第 70-72 行）

**问题类型：**
- 表头列数与数据行列数不一致
- 数据错位（如"年"字放到第二列）
- 表头重复
- 末尾冗余空列

**整改要求：** 表格须符合标准 Markdown 格式，表头列数 = 对齐行列数 = 数据行列数。

## 七、校对工作流

1. **基准确定**：以 `content/pdf-documents-formatted/` 中 PDF 格式化版为基准
2. **逐段对比**：将网页版与 PDF 版逐段对照
3. **修正差异**：
   - PDF 中有、网页中无 → 补充
   - 网页中有、PDF 中无 → 确认是否为第三方内容，若是则移入"编者注"
   - 格式不一致 → 按本规范统一
4. **确认与签名**：校对完成后标记状态
