# 微信公众号历史文章抓取脚本 - 验证清单

## 功能验证

- [ ] **文章列表抓取**：能正确获取公众号历史文章列表，包含标题、发布时间、链接、阅读量、点赞数等关键字段
- [ ] **分页遍历**：能自动遍历所有分页，抓取全部历史文章，无遗漏
- [x] **去重机制**：基于文章 ID/URL 去重，结果中无重复文章
- [ ] **文章内容抓取**：能提取正文文本、图片资源、作者信息、原创标识等完整内容
- [ ] **图片下载**：文章中的图片能全部下载到本地，按文章分目录存储
- [x] **JSON 存储**：抓取数据以 JSON 格式结构化存储，格式合法、字段完整
- [x] **CSV 导出**：文章元数据可导出为 CSV 格式，能用 Excel 正常打开且无乱码
- [x] **增量更新**：增量模式下仅抓取新增文章，不重复抓取已有内容
- [x] **时间范围筛选**：支持按开始/结束日期筛选抓取范围
- [x] **断点续抓**：抓取中断后重新运行，能从上次位置继续，已成功文章不重复

## 稳定性验证

- [x] **频率控制**：请求间隔在配置范围内，无短时间大量集中请求
- [x] **三档频率预设**：gentle / standard / fast 三档预设可正常切换且参数正确
- [x] **网络超时重试**：网络超时场景下自动重试，指数退避策略生效
