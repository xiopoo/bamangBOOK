# 腾讯 VS 泡泡玛特：平台与爆款的关系是什么？

> 投资向善 · 方伟看10年 · 2026-05-31 07:00

> [原文链接](https://mp.weixin.qq.com/s/XmaPKAmngXI5T4vAe3yrJA)

“苹果很难得，focus（聚焦）在自己做的事情上。苹果有利润之上的追求，**就是做最好的产品**。苹果文化的强度很强，有严格的‘Stop Doing List’，**一定要满足用户，一定做最好的产品。”**  

——段永平

昨天的文章《[泡泡玛特会不会成为邮票、NFT、球星卡、呼啦圈、电子宠物？](https://mp.weixin.qq.com/s?__biz=MzYzNDc3NzYzNw==&mid=2247484898&idx=1&sn=2db553a2ec077aed8149533d59f8462f&scene=21#wechat_redirect)[](https://mp.weixin.qq.com/s?__biz=MzYzNDc3NzYzNw==&mid=2247484898&idx=1&sn=2db553a2ec077aed8149533d59f8462f&scene=21#wechat_redirect)》留了一个问题：

“我最近在思考的问题是腾讯游戏/微信QQ与《王者荣耀世界》、《洛克王国》的关系，泡泡玛特品牌与拉布布、Molly、星星人的关系，抖音与汽水音乐、红果短剧、豆包的关系，**以及刀郎与他的两个徒弟，云朵与徐子尧的关系（确实很跳跃）**……到底是平台托举/流量重要还是好的产品力重要？到底是运营重要还是产品重要？”

有这个思考是我最近刷短视频，刷到刀郎新徒弟徐子尧的演唱会视频，唱的是《西海情歌》，也许是她是和声的背景，高音唱的很平庸，底下评论全是“反水”（负面）的声音，纷纷喊云朵（刀郎的旧徒弟）回来唱吧。

具体的八卦背景就不展开了，我也不了解那么多，简单理解：

1、新徒弟徐子尧：人品好，前期网民喜欢，但唱功不好；

2、旧徒弟云朵：人品差（网民想象中的人品差），但唱功很好；

而刀郎你可以理解为是一个“托举人”，**他的演唱会提供了一个平台，能够吸引千万级的粉丝的注意力，成为他的徒弟并且在他的演唱会帮唱，能够得到海量的流量曝光，以及刀郎的背书信任。**

但即便如此，徐子尧在托举之下，也并没有成功，因为唱功不行。

我们再看腾讯游戏今年的 2 款主打游戏：《洛克王国》与《王者荣耀世界》（下图）

![](./assets/6549ac3ba38bec26bd52f5e88d897ff4.png)

结果就是腾讯天美工作室七年投入数十亿开发，**寄予厚望的《王者荣耀世界》公测数据惨淡（正在调改，也有翻红的可能性），但《洛克王国》却挤进了 2026 年 4 月游戏流水排行榜的第 6 名（前三都是腾讯的：《王者荣耀》、《和平精英》、《三角洲行动》）。**

具体原因，有人分析的比我专业，我贴图一下：

![](./assets/d1be0212b7a65ec35cb88bf3514f82b7.jpg)

同样地，**泡泡玛特过去半年都是在给拉布布“降温”，不想过度消耗一个 IP（毕竟都占到营收的 38%了），开始主推“星星人”**，最好的门店 C 位、首页 Banner、线下活动、联名合作全往星星人倾斜，2026 情人节 “怦然星动” 系列几秒售罄，隐藏款溢价近 7 倍；常规系列 “美味时刻” 隐藏款溢价超 15 倍。你们应该注意到了，最近星星人\*麦当劳的活动也是火热啊。

但是，2026 年，泡泡玛特推的新 IP，比如KeyA（机械少女，赛博机械风、偏暗黑、偏未来感，设计师是陈嫣冉）、Supertutu、“放学后的 Merodi”，市场反应都不算成功。

这就非常有意思，泡泡玛特的渠道还在，会员还在，门店还在，供应链还在，发售机制还在，全球热度也还在，**但新 IP 并没有因为站在泡泡玛特这个平台上，就自然成为爆款。**

这说明什么呢？平台能给你曝光，但不能替你完成喜欢，平台没有能力把一个用户不喜欢的东西变得喜欢，**他只有能力把变量减少到最低（只需要关注产品好不好），把一个本来就很好的产品放大 10 倍、100 倍，这就是平台的意义。**

**Molly、LABUBU、星星人的成功，不能简单理解为“泡泡玛特推得好”。如果只是推得好，那 Supertutu、Merodi、KeyA 也应该被推起来；同样地，《王者荣耀》、《和平精英》、《三角洲行动》的成功，也不能简单理解为微信/QQ 有流量、有关系链、有数据，如果只是这些，那今年的《王者荣耀世界》也应该能成功。**

这就是平台型公司最残酷的地方：**它可以把一个好东西迅速放大，也可以让一个不够好的东西迅速被验证，但不能让一个不好的东西爆火。**刀郎的舞台可以让徐子尧被千万观众看见，但不能替她唱好《西海情歌》，你唱的烂甚至会让观众对刀郎的品位产生怀疑；腾讯可以给《王者荣耀世界》IP、预算、工作室和渠道，但不能替它变好玩；泡泡玛特可以给新 IP 门店、会员、盲盒机制和全球分发，但不能替它变得可爱、独特、有情绪穿透力。

平台的托举非常重要，但好产品才是第一性的，因为消费者要的就是好产品，**任何违背这一点的公司，最终都会被市场毒打。**

麦当劳 CEO 有一个讲话很有意思（下图）：“麦当劳不只是一个单一品牌。我们在主品牌之下，拥有多个十亿美元级单品品牌—— 巨无霸、足三两（Quarter Pounder，台式翻译）、麦乐鸡、麦咖啡、经典薯条、麦旋风。**这些都是标志性的独立品牌资产，总价值达 170 亿美元，并与麦当劳母品牌协同运作。”**

![](./assets/91ec165f21c84fdf6f4ac2c4999fde44.png)

麦当劳这家公司比肯德基强大很多，这是我一直以来的观点，**麦当劳会把每一款产品当成一个独立品牌来长期运营**，不会允许出现一些蹭“麦当劳”母品牌的新品，这只会消耗他的品牌资产，肯德基会经常干这种事情，因为我有渠道，所以我可以把一些不够好的产品放上去卖。

也有朋友问，胖东来是不是非常厉害的零售商？

我的判别依据很简单，消费者买东西，买的是什么？**是物美价廉的好产品。一个真正厉害的零售渠道商，他的自营品放到其他渠道，照样好卖**，比如 Costco 跟山姆、奥乐齐，因为这些公司在自营品立项初衷，想的不是蹭渠道资源，把一个比某个第三方品牌更不好的东西卖给消费者，而是把一个比某个第三方品牌更好的东西卖给消费者，做不到这一点的，都不是好的零售商，比如永辉与东方甄选，他们做自营品是为了提升毛利的。

所以，胖东来有什么产品离开了他在河南的渠道，依然是很好卖的？

我简单把零售划分为 1.0 到 4.0，胖东来模式在 2.0 的阶段，而 1.0 就是最原始的状态，就是中国上一个时代收品牌上架费、中间费的商超，他们卖海量的 SKU，加价率极高，履约费用极高，服务极差，比如物美、永辉、大润发，而 2.0 就是于东来本人用自己极高的人品，把品牌上架费、中间费都去掉，合理的毛利、合理的履约、极致的服务，做成了极强的本地信任、本地口碑，又在短视频的推波助澜之下，演变成全国性的信任、全国性的口碑，他今天已经是一个旅游景点。

但只要你不是精简 SKU 的模式，你就不可能高效，你在模式创新上依然是 0，你是“人治”，不是“法治”，前者强依赖于人，而于东来这个人是不可复制的，胖东来甚至会在于东来退下去后走下坡路。

我对商业的理解就是：真正的生意还是卖产品本身，平台、系统只是降低了好产品的获得成本。

中国有没有唱歌比徐子尧厉害很多的人？当然有很多，但他没有刀郎的托举，没有刀郎演唱会这个平台，他就没有机会展示在千万级的观众面前。

中国有没有做游戏比《洛克王国》好玩的？也是有的，但它没有腾讯的发行、买量、账号体系、社交关系链和长期运营能力**，它可能连第一批用户都很难拿到，很容易胎死腹中。**

中国有没有做 IP 比星星人更能满足某种情绪的？肯定也有，但他要么搞不定供应链，要么搞不定线下的门店，要么搞不定全球化的渠道，比如 TNT 潮玩，他们团队是没有线下的基因的，组织能力是缺胳膊断腿的，而名创他们是有线下的基因，但又不懂潮玩这个生意。

有朋友留言说：“其实 22 年还是 21 年，**腾讯也有把自己的周边毛绒玩具拿去开潮玩店，都是知名大IP，角色也都是大众早就知道的，但后面基本都关店了”**

腾讯也有 IP，也有钱，但他既不懂 IP 这门生意，也不懂门店运营是怎么一回事，他不是你找个店，交租金就能运营起来的。

阿段是开过线下店，跟商场打过交道的，他是懂的（下图）。当然，不谦虚地说，我也是懂的，原因我就不说了，保密。我可以告诉你，同样一块地，有的人谈租金会谈成5%，有的会谈成15%，商场是看菜吃饭的，这居然是需要“画饼”的，你的品牌故事在不在这个商业地产商未来的规划里面？

![](./assets/cc8d6263e62b92addf4cedd21e8f1fd6.jpg)

所以，平台的本质是什么？是降低好产品被发现的成本。

刀郎降低了好歌手被看见的成本；

腾讯降低了好游戏被分发的成本；

泡泡玛特降低了好 IP 被商品化和全球化的成本；

但这句话的反面也同样重要：**平台只能降低好产品被发现的成本，不能替代好产品本身。**

**商业的第一性，仍然是产品，没有做出好产品的公司，长期是有问题的。**

最典型的还是娃哈哈与农夫山泉，他们都是 400 万渠道点位量级的王者，但是娃哈哈是失去了做出好产品的组织能力的，他一直都在复用他的渠道，在做高销售额，而农夫山泉是既有强渠道，但又不透支渠道，他不往渠道里面“倒垃圾”，东鹏这个团队比娃哈哈好，但比农夫山泉弱。

所以，我们投资腾讯、泡泡玛特，不能说他是个游戏平台、潮玩平台，就完事了。**由于用户要好的游戏、好的 IP，还必须考察团队是否有持续做出好游戏、好 IP 的组织能力，这是很多人忽略的。**

根据我的理解，腾讯游戏的平台能力是：

**1、 微信/QQ 的社交关系链**，说人话就是玩腾讯的游戏，你看到你兄弟、你表弟、你同事、你同学都在玩、能看到排名，聚在一起“搓个麻将”很方便，你就算流失了，你朋友喊你回来玩，就又被 0 成本激活了。你说能不能喊他们打桥牌呢？当然不能，除了你会打，其他人都不会，这就没法一起玩。

**2、微信/QQ 体系的全方位导流，微信/QQ、朋友圈、群聊、公众号、视频号、应用宝、支付体系，都能给游戏导流**。这个能力并不稀缺，如果有这个能力就能做成游戏，一年做 2500 亿营收，那字节跟 Meta 早就做成了，说明流量只是必要条件，不是充要条件。当然，世界范围内，有这个流量级别的，也就是腾讯、字节、Meta 三家公司，谷歌就算有搜索跟 YouTube，也偏工具、中长视频了一点，依然跟那三个没法比。

**3、数据和运营能力强**，腾讯知道用户画像、社交关系、活跃时间、付费行为，能做精细化运营，能把 ARPU（客户生命周期价值）拉得足够长。

**腾讯游戏平台不能保证每个游戏都成功，但能降低分发成本，社交关系能提高留存，但游戏本身仍然要持续产生足够的吸引力。**

泡泡玛特的平台能力是：

**直营门店、机器人商店、线上小程序、会员体系、抽盒机制、上新节奏、海外渠道、设计师生态、供应链。**

泡泡玛特的 IP 是平台里的“情绪内容”：MOLLY、拉布布、星星人这些 IP，本质上是泡泡玛特平台里的“内容”。

泡泡玛特平台是 IP 的放大器，**让一个 IP 能迅速被看见、被购买、被传播。**

所以，结论就是，**无论是腾讯的游戏生意，亦或是泡泡玛特的潮玩生意，既要求他们是一个“平台”，也要求他们具备持续做出爆款产品的组织能力，两者缺一不可，没有前者，出爆款游戏、爆款 IP 就只能看天吃饭，概率很低，而没有后者，平台也会逐渐衰亡。**
