# 泡泡玛特的组织进化能力是被低估的

> 投资向善 · 方伟看10年 · 2026-04-12 07:00

> [原文链接](https://mp.weixin.qq.com/s/QpPBQgB2OfWPdAfleF4OVg)

“我说的是王宁对商业的理解是顶级的，**从没说过他们的商业模式是顶级的。”**  

——段永平

我先解释一下阿段开头这句话，因为我听到蛮多人说“泡泡玛特的商业模式是顶级的，就是看不清 IP 的持续性如何”。

这句话其实是矛盾的，**因为“持续性”本身就是商业模式的一部分**，为什么茅台、iPhone、腾讯的商业模式是顶级的，因为我知道你 5 年、 10 年后还在喝那瓶 53°的飞天、用那台 iPhone 25？还有微信，但我们不能确定 5 年、10 年后你还是不是在买拉布布跟 Molly。

但单个 IP 的不可持续性又不意味着公司商业模式的不可持续性，**如果泡泡玛特只有拉布布跟 Molly，那是绝对不能投资的**，这跟你投资搜狐畅游一样，这 20 年来，他就只有一款《天龙八部》，净利润从2.13 亿美元 到 1.45 亿美元，再到 1.09 亿美元、8400 万美元。（畅游当年就靠《天龙八部》一款游戏撑了很多年，这种单一爆款依赖型公司，长期是很难的。 一旦爆款不再，整个公司就跟着掉下去，这跟茅台、iPhone 这种稳健型、刚需型的大单品是两码事）

**“一生二，二生三，三生万物”，任何大公司最开始都是小公司**，都是从解决一个小小的问题开始的，苹果 1976 年的时候卖的是苹果 1 组装机套件，用户是极客，英伟达最开始卖的也不是“统治级算力平台 CUDA”，而是做 PC 图形芯片，用户是游戏厂商，CUDA 是 2006 年才推出的，被知道是 2015 年后的事，真正爆发是 2023 年以后的事。

这里面的差异是什么呢？**我觉得是组织进化的能力，这跟 CEO 本人强相关的**，张一鸣说的“Develop a company as a product（像打造产品一样打造组织） ”就是这个道理，老巴也评价，比起苹果的产品，乔布斯更伟大的的作品，是苹果这个组织以及他的文化。

有的公司是搜狐（我最开始是搜狐的产品经理，多多少少还是能了解这个公司为什么做不成事情的，光是产品名字就报批了一个月），可以说这二十年下来，除了最开始的搜狐门户，可谓屡败屡战，屡战屡败，CEO 的精力不在上面，对产品没有激情，现在做老师就挺好的，你很难在你不热爱的事情上做到拔尖。

有的公司如腾讯（社交-内容-游戏-金融与云）、阿里（电商-金融-云）、字节（今日头条-抖音-TikTok-电商-豆包）、美团（团购-外卖-酒旅-闪购）、拼多多（拼多多-多多买菜-TEMU-新拼姆），从一个点开始，慢慢地生长出完整的组织能力。

以腾讯为例，腾讯最开始也是不会做游戏的，**不会做就从代理开始**，我酿不出茅台我还不能做茅台的经销商吗？

先代理《凯旋》《QQ堂》，再慢慢做《QQ幻想》《QQ飞车》《穿越火线》的运营，再到《英雄联盟》的代理与生态运营，最后才长出《王者荣耀》《和平精英》《三角洲行动》这种真正意义上的超级自研产品。你回头看，腾讯最厉害的地方，不是“一开始就会做游戏”，而是**它有能力从不会，到学会；从学会，到做强；从做强，到把这种能力沉淀为组织习惯。**

今日的腾讯已经是世界第一大游戏公司，占据了国内 46%的游戏市场份额，全球（除中国）9%的市场份额。

而且，腾讯在游戏领域，也在不断加固自己的护城河：

**在研发端、发行端、运营端都在补自己的短板，比如研发端，《三角洲行动》是跨端游戏**，从手游绕回到端游，实现全平台覆盖，这以后会成为一种趋势。再比如 4 月份刚火的《洛克王国世界》，他在补齐腾讯在二次元、开放世界这些题材里的能力缺陷，一次生，两次熟，三次滚瓜烂熟。

再比如在发行端，如何在全球多区域本地化发行是腾讯这些年的重点，游戏全球化也是腾讯的明日之星。

还有最后的运营端，腾讯核心杀招是基于社交关系链去做运营：好友排行、战队、师徒、情侣、公会等等，又或者好友得奖励、组队开黑、社交裂变，还有活动运营，大版本+节日活动+版本赛事联动，再比如商业化运营，如何精准定价、分层付费，覆盖轻氪到重氪，卖皮肤可没有那么简单。

这些东西，腾讯都是从 0 开始做的（当然，首先是有社交的流量），积累 20 年后，一年营收到了 2416 亿，海外也从 0 做到了 700 亿。

我昨晚看到腾讯高级副总裁马晓轶（游戏界的“张小龙”）在 2025年9月德国科隆游戏展的一句话很有意思，也分享一下：

“今年大家都说我们业绩好，产品也不错。但是这件事情很有趣，**我们这两年表现得好的项目，立项或者投资都发生在2021年左右**，比如《三角洲行动》《黑神话：悟空》。但2021年之后，很长一段日子里，**在座各位都说'腾讯是不是不行了"腾讯是不是犯了很多错误'**。等到2024年的时候，突然之间一切都变好了，其实我们什么都没变。"

其实网易这个公司的利润成长对我也是有启发的，当年那些我们认为很稳定的公司，比如百度、360、58新浪都快看不见了。游戏这个行业，未必像我们想象中那么脆弱，关键还是要看这个公司的组织能力跟模式。我对非腾讯的游戏公司持保留意见，观察观察。

说回泡泡玛特，泡泡玛特也不是一下子有了 17 个年销售额过亿的 IP 的，泡泡玛特今天至少已经证明了一件事：**它不是只会“卖一个爆款”，它是能把创作者签约孵化、IP、供应链管理、线下门店、机器人门店、会员运营、海外渠道、线上内容、联名营销这些东西，拼成一个系统的。（[很多真正深的护城河，表面看是品牌，其实是系统能力](https://mp.weixin.qq.com/s?__biz=MzYzNDc3NzYzNw==&mid=2247483971&idx=1&sn=81681a62d24a6a4299ff45692c7cba47&scene=21#wechat_redirect)**[](https://mp.weixin.qq.com/s?__biz=MzYzNDc3NzYzNw==&mid=2247483971&idx=1&sn=81681a62d24a6a4299ff45692c7cba47&scene=21#wechat_redirect)**）。**

我有一个不完全正确的归纳法：**一个人或者一个公司连续成功超过 3 次，他大概率不会只是运气那么简单**，雷军 2010 年即便没找到手机的机会，他也会找到别的机会，王兴、张一鸣这些人也是，只是说机会也许没有那么大而已，大机会需要天时地利人和。

潮玩肯定是属于趋势品类，但人家王宁 16 年前开始做的时候，可没有想到今天这么火（这跟叶总这种看到风口冲进去是有本质区别的），人家是从一个格子铺开始的。

王宁这个团队在过往 16 年中是证明了自己具备持续做正确的事情的能力的：

1、为什么要从啥都卖转型只卖潮玩？这是需要商人的嗅觉的，**很多人搞半天都没意识到自己踩中了“真”需求，要快速做减法。**

2、为什么要在 2018 年选择停止跟Sonny Angel合作？2015 年一度占到泡泡玛特约 1/3 销售额，但“无法掌控”，自有 IP 才靠谱。

3、为什么沿袭 Sonny Angel 的盲盒机制？放大惊喜感和复购欲望。

4、为什么要重点做线下门店？强化体验和品牌露出。

5、为什么要做做机器人门店？提高分发密度和效率，包括测试爆款、测试流量。

6、为什么要做会员运营？沉淀用户关系。

7、为什么做联名营销？扩展圈层。

**8、为什么做全球化渠道？验证跨文化传播能力。**

......

这就开始有点像腾讯做游戏了，不会做，就先代理；代理多了，学会了发行；发行多了，学会了运营；运营久了，再往研发走，把 90%的 IP 创作者签下来；研发做强了，再往全球化走。

我们可以思考一个问题：泡泡玛特目前已经让组织习得了什么能力，目前还在努力学习哪些新能力？这些新能力会让泡泡玛特变得如何？

我觉得泡泡玛特已经习得了“**把本来难以标准化的艺术 IP 做成全球标准化的商品、做成一个个系列进行长青化运营、跨品类开发（手办、毛绒、Mega、积木等）、做成线上线下渠道高效率发行、做成全球化零售”**的能力；正在努力学习的是“把 IP 做成内容生态、做成体验空间（乐园）、做成全球品牌（目前只进入了 20 个国家而要）”的能力。

简而言之：

已经会的：IP 商品化、系列运营、跨品类开发、全渠道零售、初步全球化

正在学的：内容化、乐园化、全球品牌管理、从爆款到长青 IP 运营（这需要持续证明）

王宁在 2025 年央视专访中明确提出过两大核心战略，目的是成为“世界的泡泡玛特”：**其一是以 IP 为核心的集团化**，深度运营与孵化顶级 IP 矩阵，从单一潮玩销售商转型为全球潮流文化娱乐集团。构建 "IP + 内容 + 场景" 的生态，实现业务多元化（如乐园、内容等）；其二是全球化（国际化），**通过直营门店与全球艺术家合作**，构建 "From the world to the world"（来自世界，走向世界）的双向文化输出体系。

群里有朋友说，泡泡玛特本来就擅长门店运营，做线下乐园应该很简单。

这肯定是大错特错的，如果看过华特·迪士尼打造迪士尼乐园的书你就会发现，如何让演员每一次演出都标准化（包括怎么招聘、怎么笑、露出几颗牙齿），那是非常困难的，如何让设备100%不出故障（99%是很危险的），如何从进园-出园整个体验流程，上千个体验点都让用户愉悦，这跟做门店运营显然不一样。

也正是这个原因，乐园业务是 CEO 王宁亲自带，什么业务会由 CEO 亲自带？那就是公司本来没有的组织能力，没法考核，结果非标，只能 CEO 去做。看过《因为独特》的同学就会知道，王**宁估计把全世界运营的好的乐园都实地研究过，这是一个爱琢磨细节的 CEO。王宁也说，CEO应该管最大的事+最小的事。**

迪士尼本来是一个动画电影公司，但电影是爆款驱动的，具有不确定性，迪士尼是做了乐园以后，业绩才变得平滑的（因为乐园的功能属性更强），**乐园现在也是迪士尼最大的现金奶牛。**

**问题来了，是否意味着泡泡玛特一定要做成乐园，商业模式才稳健呢？**

我觉得未必，只能说他是加分项，能做成最好，不能做成也问题不大，乐园贡献的营收不到 1%。

泡泡玛特售卖的东西跟迪士尼是不一样的，潮玩是视觉性、审美性、情绪性的东西，他是“一首歌”、“一幅画”，他不需要通过电影故事、角色世界观来打造 IP，**发布的潮玩系列本身就是他的“电影预热”。**

当然，除了“乐园化”，“内容化”也是泡泡玛特想习得的一种新能力。

但内容产业可没有那么简单，泡泡玛特曾经花了 5 年时间开发了一款游戏《梦想乐园》（下图），高度还原**MOLLY、DIMOO、LABUBU**等全系列人气 IP，2024 年发布，现在已经有一年多没更新了，基本就是失败了。

![图片](https://mmbiz.qpic.cn/placeholder)

泡泡玛特的游戏被玩家批评玩法单一、内容重复、肝度高、抽卡机制不合理，可玩性不足。

游戏的研发、发行、运营跟潮玩完全不同，不要说泡泡玛特不行，**就连互联网里公认的组织能力最强的字节都没能成功。**

游戏的核心不是 IP、角色，而是玩法好不好玩、数值与成长系统、留存设计、付费设计、社区/活动运营节奏，泡泡玛特显然不太可能掌握这么复杂的能力，现在悬崖勒马也是明智之举。

其实电影产业同样复杂，核心也不是 IP、角色，而是剧本叙事、视听节奏、情绪调度、工业化制作、宣发排片、票房口碑闭环，这些和潮玩的 “设计 + 供应链 + 渠道” 逻辑完全是两套体系。

所以，泡泡玛特找索尼影业合作是对的，过于专业的事，只能找更专业的团队，不要自己去趟这浑水。

迪士尼本身就是干动画电影出身的，后面组织又学会了建设乐园与运营乐园的能力，这两种能力都非常复杂。

相反，你让迪士尼去学习泡泡玛特怎么设计好看、好玩、情绪价值高的潮玩、怎么运营门店、怎么管理供应链，迪士尼也两眼一抹黑。

每一个组织都有自己特别擅长的能力，也有特别不擅长的，**这跟投资一样，不要随意扩展能力圈，绝大多数能力圈的扩展都是以惨痛的失败告终的，隔行如隔山。**

“内容化”能够延长 IP 的寿命，因为好内容可被反复消费，但这件事没那么简单，目前看也是“期货”，这也并不影响泡泡玛特现在的估值逻辑。

在我看来，**泡泡玛特最关键的增长还是全球化**，这基本决定了他未来 2~3 年的业绩上限。他目前在海外只落地了20 个国家，185 个门店（不含中国港澳台），亚太（不含中国）：85 家（韩国、日本、新加坡、马来西亚、泰国等）美洲：64 家（美国 60 家、加拿大 4 家）；欧洲及其他：36 家（英国、法国、德国、丹麦等）。

你要知道，泡泡玛特卖的不是白酒，不是卤味，不是中式正餐，也不是强中国口味的东西。

**它卖的是好看 、可爱 、怪诞 、治愈 、收藏、随机惊喜 、表达自我这些东西，本身就比很多消费品更容易跨文化传播。**

一个法国女孩、一个泰国女孩、一个美国大学生、一个新加坡白领，她们对“这东西摆在桌上很好看”“我喜欢这个角色”“我想买来送人”“我想挂在包上”这类情绪，其实是共通的。

至于第二个增长点，还是组织能否持续孵化/放大新 IP，运营好老 IP。

**我始终认为一个企业能售卖一种用户需要且对手无法模仿的东西就会非常牛逼**，不需要你售卖很多东西，要满足很多需求，茅台就卖 53°飞天的口味就足够了，难道需要卖茅台牌啤酒、红酒、黄酒？

迪士尼硬要做流媒体，何尝不是一种执念，本身“爆款电影引流+乐园稳定变现”就是很好的商业模式，但还是觉得自己不够稳定，要把订阅制的流媒体给做了，但你又怎么做得过 Netflix 了，人家是正儿八经的互联网公司，组织就是为流媒体而生的。**因为“沉迷于”流媒体，结果把拍电影这个最核心的能力给糟蹋了，好几年都没有爆款电影（因为钱都拿去投资流媒体，跟 Netflix 军备竞赛），幸亏去年有一部《疯狂动物城 2》。**

你说泡泡玛特有没有可能折腾一圈乐园、游戏、电影、动画后，发现不靠谱，**回过头来安心只做持续孵化/放大新 IP，运营好老 IP**，然后在国内外把店开好，就把这一件事做好、做极致呢，这完全是有可能的。

一个把乐园运营得 70 分，把电影拍成 60 分+ 十多个不温不火的 IP 的泡泡玛特，跟另一个没乐园、没电影，**但跟腾讯游戏一样，有十多款长青 IP 的泡泡玛特，我宁可要后者也不要前者。**

任何一行，做精做深做极致都是非常困难的，很多公司不是死于主业不行，而是死于对“稳定”的过度执念，但商业世界有一个很残酷的真相：**你没法用自己不擅长的能力，去修补自己最擅长业务的波动。**

**我个人的观察，一家企业能有 2~3 个能力非常强就非常厉害了，比如伯克希尔，光做保险的没有老巴的投资能力，光做投资的，没有贾恩的保险精算能力，两个能力一组合，世界就只有一个伯克希尔。**

最后，下图阿段说的“你把我这些年说过的所有东西都看几遍也许能明白”，其实我猜，他说的就是我的意思，泡泡玛特就是泡泡玛特，核心就是卖潮玩，卖好看，卖情绪的，他就不是迪士尼，这是个全新事物。
