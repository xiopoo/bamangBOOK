花了一个下午、一个晚上的时间，一口气看完了李录写的《文明，现代化，价值投资与中国》，非常有趣，是我上半年看过的最好看的书，找到了当年第一次看《人类简史》与《穷查理芒格》时的快感。因为过于精彩，在写总结的时候，还把我的知识星球给封了hh

**2、文明是什么，现代化是什么，价值投资是什么？**

《文明，现代化，价值投资与中国》，除了中国，其他三个名词都需要定义一下：

**文明，人类利用自身与环境中的资源在生存发展中所创造出来的全部成果。**书中引述了莫里斯一段很精彩的叙述：历史，就是懒惰、贪婪、又充满恐惧的人类，在寻求让生活**更容易、安全、有效的**方式时创造的，而人类对此毫无意识。但同时人也显示出了**强大的学习能力**，一旦自然条件开始提供机会，他们很快就把自然资源条件转化成自己生存发展的巨大前进动力。”

**现代化，是人类文明3.0（1.0狩猎文明；2.0农业文明，3.0科技文明），即科技文明的阶段，以现代科技和自由市场经济为主。**当然，这个定义咱们国家是不认的，我们是一个实行公有制为主，多种所有制混合经济制度的国家，政府在经济运转过程中发挥着巨大的影响力，它可以让所有的培训机构消失，也可以突然把飞机燃油附加费加到200。当然他在战略明确的领域，有着最高的效率，能够快速组织人力、物力干大事，比如说高铁、航天航空、液晶屏、航母，最近的例子是社会性疫情清零。

价值投资，价值投资有4部分内容，第一，**买股票就是买公司的部分所有权**，所以你要关注他的业务；第二，**要预留安全边界**，因为投资的本质是对未来进行预测，而我们对未来无法精确预测，只能得到一个概率，所以需要预留安全边界，你预估美团会涨到400，现在价格只有200，安全边界就很足，如果现在已经是380了，那安全边界就不够；**第三，不要被市场先生误导**，股票市场只有5%是价值投资者，另外95%是投机者，市场是所有人综合期望的平均值，你要保持独立思考，不要盲从市场；**第四，培养自己的能力圈**，你的精力有限，你不可能看懂所有行业，投资人需要通过长期的学习建立一个属于自己的能力圈，然后在能力圈范围之内去作投资，提高成功率。

**2、文明1.0、文明2.0与文明3.0**

可以直接看我简单总结的表：

|     |     |     |     |
| --- | --- | --- | --- |
| **文明** | **时期** | **特征** | **成因** |
| **文明1.0：狩猎时代** | 始于15万年以前真正意义上的人类出现以后 | 人类文明在相当长的时间里基本和其他的动物差别不是太大，人类也没有显示出任何将来会征服地球的迹象 | **气候**：恶劣的气候条件（冰川）让人类祖先只能生活在非洲靠近赤道的很有限的区域之内 |
| **文明2.0：农业时代** | 公元前9000年左右 | 马尔萨斯陷阱：始终遵循着**“上升、冲顶、衰落”**的循环规律。因为**有限的资源和近乎无限的人口**增长决定了人口增长最终只能通过非自然灾难来消化和制约。 | **还是气候**：气候变暖让人类走出非洲，最幸运的人生活在“幸运纬度带”上，也就是欧亚大陆北纬20度到北纬35度，美洲大陆北纬15度到南纬20度之间的地区。这些地区最早出现野生谷物，所以2.0文明最有可能在两河流域诞生 |
| **文明3.0：科技时代** | 1750年左右工业革命 | GDP爆发式增长，现代科技不断涌现，全球化贸易 | **大航海出现导致大西洋经济的形成**，出现了现代科技和自由市场经济 |

书中花了一大半的篇幅来续写文明的演进，逻辑性与趣味性都很强，建议直接看书，记录一段我最喜欢的句子：

“从公元前6万年，一直到公元前12000年前，人类用了几万年的时间，从非洲出发，一路占据到南美最南端，以平均每年一英里的速度遍布了整个地球。这个时候人的主要工具就是石器，交通工具就是双腿。这时还没有农业，没有畜牧业，没有其他的动物作为依靠，也没有任何其他工具，就靠着一路打猎和采集，并以很小的团队为组织一路前进。这一场人类祖先的远征，今天想起来还会令人震撼，激动人心。”

教科书上的四个字“走出非洲”，人类足足用了五万年的时间，而工业革命到现在，不过短短三百年，一对比就非常震撼，也对马斯克们探索星辰大海由衷赞美，功在当代，利在千秋。

**3、为什么3.0的科技文明，即现代化没有在中国最先诞生？**

这个问题叫做“李约瑟难题”，无数人尝试回答过，但绝大多数回答逻辑都不严谨。我之前的理解是“中国人是实用主义的，我们有技术，但没有科技，我们不爱探寻现象背后的本质，没有形成可重复的方法论，导致技术失传，因此没有诞生科技文明”。

所谓成长，其实也遵循科学的路径，就是不断推翻自己过往的认知，知道自己之前的认知存在什么漏洞。很显然，对比李录的答案，我之前的理解不过是“过程量”，是导致中国没有诞生科技文明的过程原因，但不是根本原因。实际上就不存在哪里的人种就怎样，人性是共通的，**实际上西方出现科技文明就是纯粹地逐利，逐利就是一种实用主义。**

在上表叙述文明1.0、2.0、3.0的时候，思维本质的人肯定会看出来，我在叙述文明3.0成因的时候没有探寻本质，我写的是“大航海出现导致大西洋经济的形成，出现了现代科技和自由市场经济”，那为什么西方出现了大航海，而我们没有出现大航海？要是你还回答西方人爱冒险，中国人不爱冒险，那你就没救了~

李录给出的答案是：“农业文明时代，地理位置一直是西欧的软肋。当时西方的中心虽然已经从中东转移到地中海、南欧，但是西北欧依然很落后。不仅如此，它离富足的中国和印度也非常遥远。15世纪的欧洲人对马可·波罗笔下天堂一般的中国充满向往，希望能打开与印度和中国贸易的通道。当时西方和东方的贸易通道主要是在陆地上经由中东，而此时中东已经被穆斯林占领，又因为基督教与伊斯兰教的战争很难通过。唯一能通商的是一些零散的欧洲商人，比如威尼斯商人，通商的货物主要是印度的香料。为了找到前往印度、中国的海路，西方开始了大航海时代。”

简而言之，就是想绕过陆路的穆斯林，通过海路来通商贸易，核心就是为了赚钱。而当时“地球是圆的”已经有了理论基础，西班牙王室与葡萄牙王室为了赚大钱，就愿意天使投资给达伽马、哥伦布、麦哲伦等创业者出海，一切都是利益。这是西方比中国更早出现科技文明的根本原因。

历史充满着巧合，当时一件不起眼的事件，经过层层放大后，累积时间，塑造了后面的不同，这有种蝴蝶效应的味道。

因为大航海，发现了美洲新大陆，欧洲人去了新大陆后携带了细菌，本土的原住民基本上被细菌消灭，大西洋和太平洋基本上阻挡了美洲大陆的任何外来敌人。美洲政权既无内忧，也无外患，这就让政府不需要担当除了保护私人财产之外的任何责任。而中国政权一直要对付外来游牧民族的侵略，每到财政出现问题的时候，商业阶层就成为政府的提款机，商业活动总是为国家政权的存在和目标而服务，而不是与此正相反的大西洋经济模式。

没有发现新大陆，就没有大西洋经济；没有搞清机械世界的动力原理（因为要大规模造船、要航海），就不会出现启蒙运动的思潮批判旧的思想，拥抱全新的思想；没有这种思潮，就很难出现对科学的需求；没有发展起市场经济，也很难出现职业的科学家、技术发明家来满足经济发展的需求；没有现代科学的出现，很难想象工业革命的发生；没有工业革命的出现和传播，也很难想象大西洋经济能够迅速地形成统治全球的力量。

**自由市场经济和现代科技，是当时的中国都不具备的条件**，其中最关键的是不具备自由市场经济条件，政府无法不扮演主导作用。

以上就是现代化没有在中国最先诞生的根本原因。

**4、“讲投资你就好好讲投资，花一大半篇幅讲其他的干嘛？”**

这句话是我看完书后看书评看到的评论，当时觉得哭笑不得。因为那一大半篇幅讲的东西就是投资的一部分，这句话就像是说“抛开吴彦祖的颜值，我们来聊聊吴彦祖怎么样”。

李录通过大篇幅叙述人类文明的演进，**是为了说明整个经济出现持续性的、累进性的、长期复利性的增长和发展，才出现了现代金融产品的可投资价值，这时才有可能讨论资产配置、股票和现金**。如果你在1.0的狩猎时代与2.0的农业时代去做投资，GDP都不增长，你不可能能得到好的投资回报率。

同样的逻辑，如果现在的科技停滞了，出现逆全球化迹象了，GDP不增长了，投资也就失去了意义。

投资决策思考的维度不够多，是我过去投资犯的一个错误，特别是在政府主导经济的中国，你不得不考虑更多维的因素，比如政治因素，不能只考虑公司业务层面的因素。当时我在2018年买入美团的时候，模型很简单，因为美团的外卖撑起了美团一大半的估值，按照美团的外卖模型估算未来5年的订单量，能突破1亿单/天，美团只需要一单赚一块钱，利润就足以撑起当时美团的估值，安全边界非常高，你甚至都不需要考虑美团的到店酒旅业务、美团闪送业务、打车/共享单车/社区团购/金融/快驴等等。

一切都按预测的进行，直到2021年，时代变了。美团得接受监管局的监管调低佣金比例，未来可预测的一件事情是美团得给骑手买五险一金。当时一单能赚一块钱的模型不复存在，很可能美团的外卖就成为无利润的国民级基础设置服务，对美团只有引流价值，壁垒虽然依旧很高，但利润不再性感。于是，清仓就成必然了。

总结得有些零碎，还是希望大家看原书，这篇文章只是给大家种下一颗种子，要发芽还是得看书，然后实践。
