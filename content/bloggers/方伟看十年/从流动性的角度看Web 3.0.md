# 从流动性的角度看Web 3.0

> 方伟Rey · 方伟看十年 · 2022-04-23 12:16


![](./assets/7f854d4cd830cb1a30ac48bff9730257.png)

流动性（Liquidity），一个资产能够**以合理的价格**顺利**变成现金**的能力，评估的指标是**卖出它的时间**和**卖出的价格**是否合理。

举个例子，有一只股票，正常价格应该是20元，但是因为某个机构执行投资策略，大量的抛售这只股票，结果因为市场上买的人很少，这只股票的只能以10元的价格卖出，还有一些甚至卖不出去。那我们可以说，**这只股票流动性很差。在一个相对缺乏流动性的市场中，一项资产必须打折才能快速出售。**现金或者说法定货币被普遍的认为是流动性最高的资产，因为它可以**最快速、最容易地转换为其他资产。**

一种「法定货币」在全球贸易市场的中的流动性，也往往代表了为其背书的主权政府在国际贸易中的地位。**美元在这一方面，当之无愧是目前流动性最高的资产。**现在俄罗斯的卢布在国际贸易中等同于废纸，流动性几乎为零，国际贸易不承认卢布的价值。

**\-1-  Web 3.0在某些场景下能够提供更好的流动性，是更优的解决方案**

理解流动性对理解Web3.0非常重要，因为现代金融交易体系都是为了更好地**提升市场交易的流动性。**而一个新系统要取代旧系统，必须在系统的目标上有更优的性能，也就是**能不能提供更好的流动性。**站在计算机科学的角度看比特币与以太坊，**他们本质上都是信息、都是比特、都是一串0与1的代码**，只不过因为区块链技术的存在，这串代码具备了独一无二性，又因为比特币与以太坊的总量是固定的，具备了稀缺性，独一无二性+稀缺性的存在，**让世界上的一部分人形成了共识，大家都认可这串代码的价值。**这串代码继承了信息的所有优势，因为他本身就是信息，**信息可全球自由流动**。而比起国家主权作为信任背书的电子货币，比特币与以太坊的控制权掌握在没有“人性”的机器里，比特币就只有2100万（更准确地说，是2099.9999万个），**永远不会滥发货币，且匿名不受监管。**

或许你会质疑，现在金融体系不能提供流动性，那是因为很多贷款人没有资产作为抵押，缺乏信用，他们这样做是为了控制风险。的确，美国2008年金融危机就是因为提供给了很多没有偿还能力的人过多贷款最后坏账造成的。但未来是否存在这样的场景，我拥有价值1000万的数字资产，包括2幅总价值500万的NFT收藏品，200万的Gamefi游戏资产，以及300万Socialfi社交资产，而此时此刻我恰好想购入一套1000万的房子，而我又缺少现金，我能否以我的1000万数字资产作为抵押，快速借贷1000万，而不需要现代金融体系一两个月的重重审核？

当然，以上只是猜想，谁知道呢，我们总会低估未来的变化。
