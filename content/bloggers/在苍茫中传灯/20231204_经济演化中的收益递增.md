# 经济演化中的收益递增

> 在苍茫中传灯 · 2023-12-04 08:31


文/姚斌  

!

​

在向松祚的“新经济学”体系中，所谓的新经济学实际上就是“复杂经济学”，更确切地说是“收益递增经济学”。这是一个完全不同于传统经济学观察视角的新经济学。它的开创者是布莱恩·阿瑟。它主要从复杂性角度考察经济体系的动态演化。在向松祚的新经济学体系中，规模收益递增是一个重要的主题。  

1

新经济学特别关注偶然性、不确定性、收益递增、路径依赖、锁定效应、赢家通吃、马太效应、摩尔定律等真实世界的真实现象。它以此为工具让我们有望洞悉自然力量挑选成功者或失败者的秘密。经济复杂性研究者塞萨尔·伊萨尔戈说，也正是因为对收益递增这一理念的接纳，经济学家布莱恩·阿瑟才能摆脱经济学对复杂性的恐惧，从路径依赖的角度分析收益递增的成因，开辟出复杂经济学新的天地。

布莱恩·阿瑟在《经济学中的自增强机制》中系统的阐述了“规模收益递增+路径依赖+偶然性”的自我强化机制。他分析了这种机制的一些标志性特征。

（a）经济体系的演化往往有多条路径，但最终哪条路径取胜获得垄断优势，事先完全不确定的，也是不可预测的。

（b）那些技术上更好或更有效率的方案并不一定是最终获胜的方案，获胜者可能是次优或低效率方案，最优或高效率方案可能因为运气差而被淘汰出局。

（c）一旦某个方案或公司被市场选中或者某些难以预料的偶然性让这个方案或公司撞大运被市场所定，那么著名的马太效应或收益递增就会发挥巨大威力，迅速建立起强大的进入堡垒，让竞争对手望而却步，最终形成对市场的垄断。

（d）偶然性事件决定了路径依赖，路径依赖决定了规模收益递增。某些不可预知的偶然性事件决定了某个公司、某个技术方案、某个产品或服务在众多竞争对手中突显出一点优势。规模收益递增或马太效应迅速锁定这个优势，领先者充分利用这个优势来强化市场领先地位，从而形成自我强化机制，最终实现赢家通吃。这就是经济动态演化的正反馈机制。

经济动态演化的正反馈机制的核心正是规模收益递增。在向松祚先生看来，规模收益递增指的是，当所有生产要素等比例增长或者不成比例增长时，产出将以更高的比例增长。摩尔定律就是经典的案例。摩尔定律是英特尔创始人戈登·摩尔提出的一个假设：集成电路上可容纳的元器件数目，大约每个18~24个月会增加一倍，性能也将提升一倍。神奇的是，这个预言竟然就像一个真实存在的定律一样，一直引领着全球IC ( 集成电路 ) 和整个IT ( 信息技术 ) 行业的发展。

技术和知识含量越高的行业，其发展演化就越符合规模收益递增和技术指数式增长规律。因此，我们可以看到软件、芯片、人工智能核心技术领域的垄断和锁定现象最为普遍。随着人类社会经济日益迈向知识经济或智能经济时代，规模收益递增和技术指数式增长规律日益成为支配人类经济体系的主要规律，这是一个划时代的转变。雷·库兹韦尔在《奇点临近》中指出，服从加速回报定律 ( 即技术指数式增长规律 ) 的技术领域非常广阔，其中摩尔定律则是最著名的，也是最广为人知的技术指数增长现象。

迈克尔·马隆在《三位一体：英特尔传奇》一书中用了大量篇幅深入阐释摩尔定律：我们所有人的潜意识里都已经吸收了摩尔定律的基本思想，知道我们确实生活在一个超乎想象的、永不停息的急剧变化的世界。今天，全球经济建立在摩尔定律的基础之上，摩尔定律决定了现代人类的节拍。尽管戈登·摩尔早期曾经预测变化的速度会在20世纪70年代后期缓慢下来，但摩尔定律却一直持续发挥作用。直到大约2005年，摩尔定律那条曲线开始弯曲成为一条直线，这预示着，尽管摩尔定律在过去半个世纪给世界带来全方位的革命性的变化，但真正的变革还在未来。未来历史将会认为这个奇迹是全部人类历史上最伟大的成就之一，摩尔定律则预言了这个奇迹的发生。

2

规模收益递增的本质是人的无限创造性，只要沿着某个特定方向持续不断的努力，就会持续降低技术、产品和服务的成本，并遵循一条持续向下的成本曲线。后继者如果试图模仿先行者的路径，则面临不可逾越的先行者优势壁垒，因为规模收益递增的背后机理是先行者持续不断的学习、经验或诀窍的积累，以及相关领域的专利权保护。从这个意义上说，技术追赶并没有弯道超车只有变道超车，也就是采取不同的技术方案来替代先行者的方法，从而抢占先行者的市场份额。

类似于日本、德国、瑞士的高精尖制造；英特尔和高通的芯片技术；微软、谷歌和苹果的操作系统；通用电气和罗罗公司的飞机发动机制造技术；罗氏、辉瑞、默沙东等制药巨头；谷歌的搜索引擎和相关技术等等，就是正反馈机制或规模收益递增形成的强大坚固的技术壁垒，让竞争对手望而却步，让领先者持续领先，只有出现新的替代技术和产品，才有望打破它们的赢家通吃或垄断者的地位。

经济体系到处呈现出各种突变或令人惊奇的现象，20世纪90年代后期，众多年轻企业家创建公司以争夺互联网门户的商机，过了不到5年时间，互联网门户就被三五家公司垄断。世纪之交时，搜索引擎、电子商务、社交媒体刚刚从地平线露出曙光，各国都有数十家乃至数百家公司群雄逐鹿，也是不到5年的时间，无论是搜索引擎、电子商务还是社交媒体，都形成了寡头垄断甚至独占的产业格局。百度基本独占了中国搜索引擎市场，谷歌则垄断了英语世界和许多非英语国家的搜索市场。亚马逊成为美国乃至全球电子商务的寡占巨头，阿里巴巴则形成一个庞大的商业生态体系或商业帝国，前5家电子商务公司基本上垄断了全球的电子商务市场。美国社交媒体只剩下脸书、领英等少数几家企业，腾讯则成为中国社交媒体无可争议的垄断者。阿里巴巴和腾讯的移动支付平台基本上瓜分了全中国移动支付市场，而且将势力范围迅速扩展到海外。

从复杂性和创造性角度考察经济增长，首先是最终消费品和服务的多样性不断延伸、不断丰富；其次是每一种产品、每一种生产要素、每一种物质的用途皆有无限多样的可能性，它们的用途不断被延伸、被拓展、被开发出来；最后是产品、服务、技术之间的结合或组合不断被创新，从而产生出全新的商业模式、产业业态，由此又创造出无限多样的产品、服务和技术，同时刺激生产组织和生产方式的多样性和革命性变迁。

这种超乎想象的无限前景，是任何人、任何数学模型都无法准确预测的。创新或创造的一个最基本方式，就是将不同的思想、不同的模式、不同的产品、不同的服务、不同的技术、不同的组织、不同的人等进行组合，从而导致完全意料之外的效果。这就是创新的本质，也就是经济增长的本质，它是经济体系演化过程从本质上无法预测的基本原因。

正如圣塔菲研究所的创始人之一、物理学家菲利普·安德森所说的“多就是不同”。复杂性必然意味着多样性，多样性必然意味着复杂性，多样性和复杂性就意味着无限可能性和无限创造性。雷·库兹韦尔著名的“奇点”预测，正是基于人类多样性技术之间的广泛融合，必定带来超乎想象的新技术革命。以研究竞争战略驰名世界的迈克尔·波特说，竞争就是创造不同，即竞争的本质不是创造同质化的产品和服务，而是创造多样性和独特性，是创造不同，也就是史蒂夫·乔布斯所说的“非同凡想”。

3

从经济体系演化的角度来看，新的商业模式、新的产品、新的服务和新的公司随时随地都在涌现、进入死亡或退出。因此，没有人能够预测哪家公司会成功存活10年还是20年，也可能一年之内就会就会销声匿迹。多年前默默无闻的新创企业在数年之内就会迅速崛起成为全球性巨头，如谷歌、脸书、阿里巴巴、腾讯、特斯拉等，曾经不可一世的商业王国可能在一夜之间分崩离析，如雷曼兄弟、美林证券、AIG等。创造性毁灭的速度如此之快，即使是那些长期密切观察人类经济体系演化的人也觉得始料不及。

而伟大的企业，尤其是居于产业链最上游或居于产业圈最核心位置的企业，都催生或衍生出一个庞大的产业圈或产业链。无数企业随之应运而生。像苹果这样的智能手机发明者，不知催生出多少个手机整机和配件的生产商、供应商、专卖店、修理店，以及数之不尽的相关产品和服务。App Store 是绝对是一个天才的发明，是商业历史上的一个奇迹，仅此一项就足以显示史蒂夫·乔布斯及其团队的非凡创造力。一个新产业或新产业的领袖企业的伟大作用，为产业和经济演化提供了新的催化剂和聚合点。它们就像核裂变链式反应。一旦引爆，就会快速创造出一个庞大无比的产业圈或产业链。

从人类经济体系的演化历史来看，“公司”是演化速度最快的“新物种”。凯文·凯利在《失控》中写道，近年来复杂性理论的研究人士也已意识到，任何活系统——经济体、自然生态系统、复杂的计算机模拟系统、免疫系统以及共同进化系统——都具有摇摇欲坠的显著特征。当它们保持着埃舍尔式的平衡态——处在总在下行却永远未曾降低过的状态时，都具有那种似是而非的最佳特征——在塌落中平衡。

一个生态或生物群落里，最初主宰该群落的生物会不断失去或消失，新的生命会不断涌现出来，成为生态群落或生物链的主宰。一个大型企业破产关闭退出市场，总有新的企业迅速长大，成为产业圈或经济圈新的中心节点。硅谷的发展历史清楚地告诉我们产业生态圈的演化过程和内在机制。

今天全世界的高科技产业链或产业圈，就是以硅谷为核心和支点而建立起来的。过去半个世纪以来，全人类最前沿和最惊人的科技创新，绝大多数都源自硅谷，然后波及或影响到整个地球。从互联网、个人计算机、智能手机到人工智能、机器人、大数据、云计算、物联网、虚拟现实、区块链等，硅谷的每一项重大观念和科技突破都引发了世界性的产业震动和冲击波。每一项产品和服务都是颠覆性的突破，之后迅速引发世界性的学习、模仿、抄袭和跟随，全球产业链和产业圈由此诞生。

互联网经济标志着信息时代的来临，互联网经济的本质特征是规模收益递增规律。每增加一个节点，任何所有节点的效能和作用就会增大，一切创造皆符合收益递增规律。任何一种精神、思想、理念知识的创造和传播都符合收益递增规律。分享越多，思想和知识的价值和效能就越大。智能手机、计算机的操作系统、社交网络以及电子商务平台等皆符合规模收益递增规律。所有信息科技都具有规模收益递增的特征。规模收益递增必然导致锁定效应、赢家通吃和马太效应。在信息科技时代，体量越大的公司，其收入和利润的增长速度反而越快。这是非常奇特的规模收益递增现象。

凯文·凯利认为，进化系统的本质是一种产生永恒变化的机制。永恒的变化意味着持续的不平衡，永远处于即将跌落的状态。它意味着对变化做出变化。当然，绝大多数企业或公司的行为是跟踪、模仿、调整、改善，只有极少数企业行为称得上是真正的颠覆性创新或创造性破坏。每个时代称得上真正具有颠覆性和革命性创新的公司寥若晨星。一个真正具有颠覆性和革命性企业的崛起，就代表一个新兴行业的诞生。

通用电气和西门子的创立标志着电力时代的来临；奔驰和福特标志着汽车时代的来临；标准石油和荷兰壳牌标志着石油时代的来临；IBM标志着大型计算机时代的来临；索尼和松下代表着现代家电时代的来临；仙童半导体和英特尔的诞生标志着信息时代的来临；苹果公司的诞生标志着个人计算机时代的来临；微软公司的诞生标志着现代软件业的诞生；谷歌、亚马逊、脸书标志着互联网时代的来临；iPhone标志着移动互联网时代的来临；特斯拉标志着电动汽车时代的来临……

确实，有时可以说，只有时代的企业，没有伟大的企业。伟大的企业也会过眼云烟。人类经济体系的本来面目就是一个具有无限创造性、持续动态演化的生命体系。从生命体系的角度来考察和理解人类经济体系是最根本的经济学范式转变。
