# 跟随科学的脚步

> 在苍茫中传灯 · 2023-08-27 08:01

真正的投资者总是试图更好地了解卓越的公司是如何获得成功的。因为卓越的公司定义了现代世界，并带来了丰厚的财务回报。对此，我们也一样，试图理解卓越公司的成功密码。制造业巨擘辉瑞公司就是这样的一个案例。辉瑞董事长兼CEO艾伯乐在他新作《科学的胜利》中，就叙述了辉瑞在整个新冠疫情期间全力以赴研发mRNA新冠疫苗的过程。在这个过程中，让我们看到了一家卓越的公司是如何用科学的力量应对不确定性的未来，突破极限、挑战不可能，从而走向卓越乃至“伟大”。

1

辉瑞的成功故事由三大因素共同促成：科学的力量、生机勃勃的企业对社会的意义以及人类智慧的巨大潜能。早在2018年，艾伯乐就被任命为辉瑞的首席运营官。那时他的关注点是增长，将“增长绝非机缘巧合，而需主动创造”奉为左右铭。推动制药行业增长的唯一方法，就是对患者的生活做出积极的改变。为此，必须将辉瑞转型为一家以患者为本的公司，一家坚持科学与创新的公司。

2019年，艾伯乐出任辉瑞董事长兼CEO，开始了他的“奇迹之旅”。他立即启动了公司历史上最激动人心的改革，致力于领导辉瑞实现“为患者带来改变其生活的突破创新”，改善全球民众的健康状况。他要将辉瑞打造成一家“最好”而不是“最大”的公司。他深知，要想让一家优秀公司跻身伟大公司之列，就要依靠对自身成功业务进行优化的能力，以及开启新篇章和拓展更新、更广前景的魄力。最终的目标是转型为一家专注于科技创新的公司。

在艾伯乐看来，公司如果要商业转型，不能仅依靠改变企业投资组合或资产配置来完成。要想成功，还必须改变公司的文化。辉瑞必须成为一家能够适应风险的公司，并有动力采取大胆行动，达成使命，兑现对患者的承诺。

随着全球人口老龄化和城市化进程的发展，人们对创新型医疗解决方案的需求将会持续增长。我们需要的是一种通过大胆思考和打破常规来促进创新的文化。随着医疗保健服务成本的增长，医疗差异的鸿沟或许会继续拉大。因此，我们需要一种对社会需求更加敏感的文化，并努力于达成使命。但是，新的文化无法在真空中创造。我们无法要求商学院的顾问给出最契合的商业文化供我们选择。一家公司制胜的文化，取决于它对人类的贡献、公司本身关于成败的经验传承、未来10年的挑战和机遇、所处的环境以及许多其他因素。我们需要对拼图的所有拼块进行评估，设计出属于自己的制胜文化。

辉瑞之所以存在，是因为社会需要并要求辉瑞“为患者带来改变其生活的突破创新”。针对是否该用药物或疫苗来描述这一突破，辉瑞展开了大量的探究，但最终还是决定使用“突破创新”一词，因为这个词的涵盖范围更广，也更加掷地有声。随着技术之间的界限变得模糊，只探讨药物或疫苗似乎很有局限性，无法映射出科学探索近10年的现状。在短短几个月后，辉瑞启动了一项向公司员工和全世界介绍全新企业文化的企划，且只用了4个简单的词语：勇气、卓越、平等、欢乐。

2

在2019年1月谈到“为患者带来改变其生活的突破创新”时，没有人能想到，不到两年后，辉瑞的转型竟然能够改变千千万万人的生命。而今，辉瑞已经研发出世界第一款安全有效的新冠疫苗，并在多国获批。辉瑞的成功并不是靠运气，而是因为他们做好了准备。辉瑞人在过去几年中认识到，敢于心怀高远，做出艰难而看似为违背常理的决定，不仅不会被禁止，而且会受到鼓励；践行卓越，不是为了自身利益，而是为了服务患者；平等和缩小医疗差异虽然不是其他公司关注的重点，但却是辉瑞目标的重要组成部分。辉瑞带给世界的美好，反过来能够令他们感到欢乐、自豪以及激励他们对目标的诚挚奉献。

辉瑞选择了mRNA技术来开发新冠疫苗，并不是一个显而易见的选择。这种新型mRNA疫苗能够教会身体制造自己的疫苗，疫苗则等同于一系列关于如何自保的指令。辉瑞一直对RNA抱有浓厚的兴趣，其原因在于RNA能够对变化迅速产生反应并具有稳定性。比起传统的疫苗技术，辉瑞更偏爱这种技术的灵活性。如果某种新出现的病毒株没有被当前疫苗彻底覆盖，这种技术便可以灵活改变疫苗中的RNA序列，从而有可能对新病毒株起效。

然而，这却是一个巨大的赌注。艾伯乐的怀疑是以事实和理性思考为依据的。首先，这项技术虽然很有前景，但尚未得到证实；其次，辉瑞必须与德国生物新技术公司达成协议，而这个过程通常需要数月时间才能完成；其三，由于德国新生物技术公司规模较小，辉瑞可能要承担所有研发和生产成本，一旦失败，就要承担巨大的损失，而一旦成功，则必须与对方平分收益。

由于mRNA是合成物，不含活病毒颗粒，因此这是一种更加精准直接的产品。这样一来，辉瑞的团队便可按需多次使用这种疫苗，不仅能在新冠病毒感染大流行期间如此，在日后由于病毒突变而需要对疫苗进行加强时也是如此。传统疫苗的设计可能需要几个月的时间，而这种mRNA疫苗的设计只需几个星期。辉瑞在这项技术积累了丰富的专业知识。他们认为这项技术已经成熟到可以用来制作疫苗的水平。而一旦有了mRNA，辉瑞就可以快速推进研发工作，但疫苗在配送期间必须冷冻才能保持稳定。

在生命科学中，疫苗学家扮演着特殊的角色，医学领域的所有医生和生命科学都致力于为大众服务。但是，疫苗学家之所以选择这一领域，是为了专门研制疫苗，而且不遗余力地保障大众的健康。他们通常比较保守。与用于治疗疾病和慢性病的药物不同，疫苗的功效在于预防。与大多数医生相比，疫苗学家必须用不同的方式衡量风险和收益。在研究项目的安全性和有效性上，疫苗学家面临着层层审查，因此在选择上也非常保守。

3

成功研发一款疫苗的难度超过了我们的想象。在正常情况下，一款疫苗的研发要经历数年时间。在这个过程中，许多尝试都会以失败告终。以艾滋病病毒为例，科学家已经花了几十年时间，但仍然没有研制出一款有效的疫苗。从发现、开发、批准、生产到销售，在被注入人体之前，新款疫苗必须遵循一个高度规范的过程。这个过程需要耐心和毅力。

科学家会因为基础科学的某个理论而在研究室研究多年，然后才有权开发出几款“原型”，进行深入测试。经过一个反反复复且往往长达数年的过程，这个原型才能在试管中和动物身上测试，然后被送回实验室进行调整，从而进一步完善理论，并从理论上提升感知的有效性或安全性。经过这些调整之后，这些原型会经历一次又一次的测试，以期找到最佳候补选项。然后进行临床前和病理学研究、反应原性研究、病毒中和性研究、免疫原性研究、疗效研究等各种高度专业化研究。

一般而言，评估不同原型所需的测试和试验上不存在，因此也需要研究人员在检测原型之前创建出来。每当研究人员发现老鼠发烧或弱病毒中和等不愿看到的现象，就会进行额外的试验来理解这些影响，并提出修改原型的建议。当然，原型必须具有化学或生物上的可扩展性。这意味着一旦成功，这些原型便可投入大规模、高质量的生产；而如果失败，这一颇有前景的原型开发便可能化为泡影。这种开发并非总能成功。很多时候，之所以停止开发颇有前景的分子，是因为这些分子无法在实验室外进行大规模复制。如果这些实验室的临床前研究取得了成功，就能确定一些候选疫苗，让我们有充分的理由相信它们的安全有效性。

然后，实验人员便开始在人体上进行临床试验。首先要进行研究的是确定合适的剂量。这就进入第一阶段临床试验，又称为剂量递增研究，目的在于寻找任何可能安全隐患的信号。随着逐步增加剂量，直到找到一个安全并满足替代标准的水平。若做不到，便会重新设计候选方案，用新的候选疫苗重新开始第一阶段临床试验。这个过程通常会重复数次，直到取得成功或决定终止项目。

如果第一阶段临床实验取得了成功，就会对最佳候选疫苗进行第二阶段的临床试验。此时，通过不同的治疗方案对候选疫苗进行试验。并且，尝试多种组合，不断测量替代终点的安全性和有效性，直到确定最优选项。第三阶段临床试验至关重要，需要获得监管部门的批准。这些研究必须遵守由FDA和欧洲药品管理局等管理机构制定的一系列极其严格的规则和标准，不违背道义操守和科学的严谨性。通常情况下，这些研究要持续很长一段时间，由数千名志愿者参与和数百名独立医师或调查员监测。

在这些研究中，实验人员通常会设立实验组与对照组。实验组的参与者将注射处于评估环节的疫苗。对照组的参与者则注射另一种已经注册并符合当前护理标准的疫苗或安慰剂。如果两组阳性病例数的数据差异未达到统计显著性，应继续进行研究；如果试验组和对照组之间数据差异无法达到统计显著性，就证明实验无用，应停止研究；如果疫苗有效性已达到统计显著性且不存在安全性问题，就证明达到药效，应停止研究。

疫苗的有效性和安全性通过第三阶段临床实验得到证实后，就会执行接下来的两个步骤。（a）准备备案材料，包括监管部门审批需要的所有数据。典型的疫苗初始申请材料有数千页之多，每一页都需要为确保准确性进行质量控制，而且这一过程也十分耗时。（b）为新疫苗的工业化生产做准备，包括订购原材料，在某些情况下还需要购置通常价值数亿美元的专业设备。

FDA规定，疫苗的有效性需达到50%才能获得批准。团队设计的试验，则能让疫苗的有效性达到60%。辉瑞的内部标准高于官方的标准。对于紧急使用授权申请，FDA通常要求2个月的安全数据，全面批准则需要6个月的安全数据。无论官方如何要求，辉瑞都会对这些参与者进行为期两年的追踪调查。辉瑞在数学领域的专家进行了统计分析表明，想要证明该有效性水平 ( 60% ) 的统计显示性，他们至少需要164个新冠病毒感染事件 ( 即感染新冠病毒的参与者 ) 。一项拥有1万~1.5万参与者的研究，得出这些结果或许需要不到1年时间，需要时间取决于疾病/感染的发生率。辉瑞扩大了研究规模，招募了3万名参与者，以便更快地积累新冠病毒感染事件。后来，他们又将这一数字增至超过4.6万名。一项如此规模的研究，通常要用到40~60个科研场地。辉瑞设置了120个科研场地，以便更快地招募到参与者。后来，研究人员又把科研场地的数量增至153个……

辉瑞的故事告诉我们，在事态最多变难料的时候，要相信科学的力量。辉瑞利用科学，加速了胜利的实现，因为科学的胜利也是我们每个人的成功。在捍卫“为患者带来改变其生活的突破创新”这一使命上，辉瑞一丝不苟，十分严谨。这就启示我们，无论科学把我们带到何处，都要跟随科学的脚步。辉瑞的故事让我们知道了“卓越”从何而来。
